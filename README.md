# LEARN BASIC EXPRESSJS

**A simple RESTFULL API using [ExpressJS]('https://expressjs.com'), [MySQL](https://mysql.com), and [Sequelize]('https://sequelize.org).**

List Routes

	GET /products
	GET /product/:id
    GET /products/keyword={name}
    POST /product
		- name
		- price
		- description
    PUT /product/:id
    DELETE /product/:id
